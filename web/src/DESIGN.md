# InfoMerge Design System

## Mood

## Colors
- bg-primary: `#0d1117`
- bg-secondary: `#161b22`
- bg-tertiary: `#21262d`
- border: `#30363d`
- text-primary: `#e6edf3`
- text-secondary: `#8b949e`
- accent: `#58a6ff`
- accent-glow: `#1f6feb`
- success: `#238636`
- warning: `#9e6a03`
- error: `#f85149`

## Typography
- body: `-apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`
- mono: `"JetBrains Mono", "Fira Code", monospace`
- h1: 24px / h2: 20px / h3: 16px / body: 14px / small: 12px

## Components
- Cards: bg-secondary, 1px border, 8px radius, padding 16px
- Buttons: accent bg, white text, 6px radius, padding 8px 16px
- Inputs: bg-primary, border color, focus ring accent-glow
- Tags: bg-tertiary, mono font, padding 2px 8px, 4px radius
